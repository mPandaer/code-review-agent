# AGENTS.md

本文件是项目规则入口。具体规则已拆分为独立文件，按场景渐进式加载。

## 规则文件

| 文件 | 职责 | 加载场景 |
|------|------|----------|
| `specs/001-导师角色定义.md` | 导师角色定义 + 协作方式 + 执行边界 | 开发 Code Review Agent 时激活 |
| `specs/002-项目背景.md` | 项目背景 + 设计文档索引 | 讨论项目设计、需要上下文时激活（与 `001-导师角色定义.md` 配合使用） |
| `specs/003-文档同步规则.md` | 设计文档写作规范 + HTML 索引同步 | `specs/business/md/` 目录下文档变更时激活 |

## 加载方式

- **开发场景**：同时加载 `specs/001-导师角色定义.md` + `specs/002-项目背景.md`
- **文档同步场景**：仅加载 `specs/003-文档同步规则.md`
