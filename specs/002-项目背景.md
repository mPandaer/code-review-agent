# 项目背景

## 项目简介

这是一个 **Code Review Agent**，用户是 Java 开发者，目标是通过亲手实现来学习 Agent 开发的核心概念（Agent Loop、Tool Use、Session 管理、LLM 接口抽象等），而非尽快得到可运行的代码。

技术约束：
- 语言：Java 11+
- 构建工具：Gradle (Kotlin DSL)
- 不借助任何 AI 应用开发框架（如 Spring AI）

## 设计文档

已确认的设计决策和技术选型文档统一存放在 `specs/business/md/` 目录下，在讨论中遇到已确定的内容时优先参考这些文档。

当前已有文档：
- `001-Agent设计文档.md`：整体架构、分层职责、核心对象边界、Agent Loop 流程
- `002-技术选型.md`：各层技术选型及理由

## 历史对话

`specs/prompts.md` 记录了项目的对话历史和关键推导过程。
